JavaScriptチートシート

Array メソッド

  • concat　　　2 つ以上の配列を結合し、結合した配列のコピーを返します
  • push　　　　配列の末尾に新しい要素を追加する
  • pop　　　　　配列の最後の要素を削除
  • indexOf　　　要素の配列を検索し、その位置を返します
  • lastIn­dexOf　配列の末尾から要素を検索し、その位置を返します
  • slice　　　　　文字列や配列などからデータの一部分だけ取り出せる
  • length　　　　配列の要素の数を取得する
  • shift　　　　　配列の最初の要素を削除し、その要素を返します
  • includes　　　特定の要素が配列に含まれているかどうかをtrue / false で返します

Number メソッド

  • toExponential(x)　　　　数値を指数表記に変換します。
  • toFixed(x)　　　　指定した少数点以下の桁数で数値を四捨五入してくれます。
  • toString()　　　　文字列でないものを文字列に変換するとき
  • toPrecision(x)　　　　Numberオブジェクトを指定された精度で表した文字列を返します。

Date メソッド

  • getFullYear()　　　　指定された日時の年を返します（4ケタの年）
  • getMonth()　　　　「月」を表す0~11までの値を返す。 指定された日付の「月」返す。0 が年の最初の月を示す(0 ~ 11)
  • getDate()　　　　　指定された日付の「日」を返します。(0 ~ 31)
  • getHours()　　　　指定された日時の「時」を返します。(0 ~ 23)
  • getMinutes()　　　　　　指定された日時の「分」を返します。(0 ~ 59)
  • getSeconds()　　　　　　指定した日時の「秒」を返します。(0 ~ 59)

Mouse Event

  • click　　　　　　　クリックされたときに発生
  • dbclick　　　　　　 ダブルクリックされたときに発生
  • mousedown　　　　 マウスボタンが押されたときに発生
  • mousemove　　　　 マウスポインタが要素の上を移動したときに発生
  • mouseover　　　　 マウスポインタが要素に乗ったときに発生
  • mouseout　　　　　マウスポインタが要素から離れたときに発生
  • mouseup　　　　　マウスボタンが離されたときに発生
